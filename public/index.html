Twitter API Toolkit for Google Cloud

Service is Running

This service acts as a filtered stream adapter connecting Twitter API v2 to Google Cloud services including BigQuery and PubSub.

Available Endpoints

  • /stream - Stream Controller
    Manages the connection to Twitter's filtered stream.
  • /rules - Rules Management
    Add or remove filtering rules for the stream.
  • /api - General API
    Access trending data and other utility functions.
  • /token - Token Management
    Handles authentication tokens.
